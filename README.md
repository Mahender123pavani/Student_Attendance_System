# Student Attendance App

This small project includes a database helper and tests.

Environment variables (recommended):

- DB_HOST (default: localhost)
- DB_PORT (default: 3306)
- DB_USER (default: root)
- DB_PASSWORD (REQUIRED)
- DB_NAME (default: attendance_system_db)

Running tests (uses virtualenv in this project):

```powershell
C:/Users/mahen/Downloads/Student_Attendance_App/venv/Scripts/python.exe -m unittest -v
```

If you need to create a MySQL user for testing, see `create_db_user.sql` for an example.
